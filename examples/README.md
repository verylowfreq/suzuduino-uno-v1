#### 液晶パネルの駆動 (Arduino IDE)

SPI接続の液晶パネルにパターンを表示します。


#### USBシリアル通信 (Arduino IDE)

USBシリアルとして振舞います。UART2 (PA2, PA3) をUSBへそのまま流します。
